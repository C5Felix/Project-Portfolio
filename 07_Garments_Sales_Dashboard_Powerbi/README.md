# Garments Sales Dashboard (Power BI)

## Project Overview

This project presents an interactive sales dashboard built using Microsoft Power BI to analyze garments sales data. The dashboard provides insights into product performance, sales distribution, and category-wise revenue.

## Objective

To analyze garments sales data and identify patterns that help understand product demand, sales trends, and business performance.

## Dataset

The dataset includes:

* Product categories
* Sales amounts
* Order information
* Customer purchase data

## Key Dashboard Insights

* Total sales performance
* Category-wise revenue distribution
* Top-performing garment products
* Sales comparison across product segments
* Identification of high-demand products

## Tools Used

Microsoft Power BI
Microsoft Excel

## Skills Demonstrated

Data analysis
Business intelligence dashboard development
Interactive data visualization
Sales trend analysis
